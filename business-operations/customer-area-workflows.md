# Customer Area workflows

Operators should be able to tell quickly where the UI is the source of truth and where APIs still matter.

## Typical high-value tasks

- Reviewing payment activity
- Investigating exceptions
- Managing configuration and access
- Coordinating with developers on events or integration behavior

The docs should help non-developers stay productive without hiding the technical dependencies behind the workflow.
