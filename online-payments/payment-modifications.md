# Payment modifications

Post-payment actions are where engineering, support, finance, and operations start sharing the same documentation surface. This page groups the modification concepts that are otherwise easy to scatter.

## Main modification types

* Capture
* Cancel
* Refund
* Reversal
* Authorization adjustment

{% hint style="info" %}
For a docs-owner prospect, this is a good demonstration area because it shows how one documentation platform can support both builders and operators without forking the content experience.
{% endhint %}

Reference source: Adyen's live [Payment modifications](https://docs.adyen.com/online-payments/modify-payments/).
